<ww:form />

一个HTML表单.

用法示例 

<ww:form name="'myForm'" action="'submit.action'">
...
</ww:form>

<ww:form name="'myForm'" action="'submit'" namespace="'/foo'">
...
</ww:form>

属性
名字 必需 描述
name 是 HTML的name属性
value 否 作为字段值
openTemplate 否 用于标签开始的模版
action 是 HTML的action属性
namespace 否 活动的名空间
target 否 HTML的target属性
method 否 'GET'或'POST'
enctype 否 HTML的enctype属性
validate 否 是否使用客户段校验
cssClass 否 HTML的class属性
cssStyle 否 HTML的style属性
theme 否 使用的主题
template 否 用于标签结束的模版名


注意: action属性有两种工作方式. 第一个例子中展示的是最初的(也是被废弃的)格式. 新格式允许仅使用定义在xwork.xml中的活动名称(如果有名空间, 还要加上名空间). 如果需要客户段校验的话, 这是必需的
