<ww:submit />

提交按钮.

用法示例 

<ww:submit value="'Submit'"/>

属性
名字 必需 描述
id 否 HTML的id属性
name 是 HTML的name属性
value 否 文本值
align 否 HTML的align属性
label 否 用于模版中的标签文字
labelposition 否 标签的对齐方式 (left,right,center)
required 否 在表单提交时是否是必需字段
cssClass 否 HTML的class属性
cssStyle 否 HTML的style属性
theme 否 使用的主题
template 否 使用的模版名
