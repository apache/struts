<ww:text />

输出一个经过国际化处理的字符串. 它与i8ln标签联合使用. text从周围的i8ln标签定义的资源包中获取指定的文本信息. 参数值可以传递到消息中, 例如格式化日期或货币.

属性
名字 必需 描述
name 是 提取的属性名
id 否 指定时, 将输出结果使用id作为键值保存在ActionContext中
value0 否 传递消息中的参数0
value1 否 传递消息中的参数1
value2 否 传递消息中的参数2
value3 否 传递消息中的参数3

用法示例 

Accessing messages from a given bundle (the i18n Shop example bundle in this case)<br>
<ww:i18n name="'webwork.action.test.i18n.Shop'">
  <ww:text name="'main.title'"/>
</ww:i18n>

<ww:i18n id="foo" name="'webwork.action.test.i18n.Shop'">
  <ww:text name="'main.title'"/>
</ww:i18n>
<ww:property value="#foo"/>

注意: 你可以使用下面的格式替换value0..value4: 

<ww:text name="'someKey'">
    <ww:param value="'Hello'"/>
</ww:text>

或者

<ww:text name="'someKey'">
    <ww:param>Hello</ww:param>
</ww:text>

当你将HTML嵌入消息中时, 这种格式非常有效, 因为你不需要担心这部分代码可能有转码(escaping)问题.
